# Transformer from Scratch

## 概要
PyTorchを使用してGPTモデルをスクラッチで実装

- GPT-124M相当のモデル実装（768次元、12ヘッド、12レイヤー）
- Multi-Head Attention、Layer Normalization、Feed Forward等の詳細実装
